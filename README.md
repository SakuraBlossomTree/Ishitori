# Ishitori

Ishitori is an image booru browser built with React Native. Created as a fun side project, the app allows users to browse and search images from supported booru websites through a simple and mobile-friendly interface.

## Features

* Browse images from supported booru websites
* Search using tags
* Simple and responsive user interface
* Mobile-first design
* Built with React Native



```bash id="1f9hwr"

```

```bash id="6ef5m0"

```

Run on iOS:


Ishitori was developed as a personal side project to learn React Native while building a lightweight image browsing application. The primary focus was creating a smooth mobile experience and experimenting with API integration.
